Deeksha Sharma – Portfolio
Interactive personal portfolio for Full Stack Technical Lead Deeksha Sharma, focused on showcasing experience, leadership, and front‑end craft with motion, theming, and premium UI effects.

Tech Stack
Framework: React (SPA) with react-router-dom for routing.
​

Styling: SCSS (Sass modules), custom variables/mixins, responsive layouts.

Animation: motion/react (Framer Motion v5 API), subtle spring and fade transitions.
​

Effects: react-snowfall for canvas‑based hero snowfall.
​

State & UX:

React hooks (useState, useEffect) for loader, theme, and layout behavior.


Getting Started
Install

bash
npm install
# or
yarn
Run dev server

bash
npm run dev
# or
yarn dev
Build

bash
npm run build
# or
yarn build
